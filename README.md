# Premium Plumbing Solution

A high-end, animated plumbing website built with React, Tailwind CSS, and Framer Motion.

## Features
- **Premium Design**: Dark, gold-accented theme.
- **Interactive UI**: Swipeable service cards, scroll-triggered animations.
- **Performance**: Optimized with Vite.

## How to Run

1. **Install Dependencies** (if not already done):
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   This will start the server at `http://localhost:5173`.

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `src/components/`: Core React components (Hero, Services, Contact, etc.)
- `src/index.css`: Global styles and Tailwind configuration.
